


  ### Branch related information

 ### Create Branch
    
      'git branch <branch-name>'

 ### See all the branch
      
      'git branch'

 ### Switch Branch
     'git checkout <branchname>'

  Conflicts : When two branches contains different changes in the same file it raise a conflict
 and it must be resolve by 'manually'.
 ## Merging


    ## Merge changes from branchA into branchB

      'git checkout branchB'
      'git merge branchA'
   
   if there are no conflicts then it create a merge commits.
